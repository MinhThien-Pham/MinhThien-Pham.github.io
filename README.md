# Portfolio Website

This is a portfolio website built with React, TypeScript, and Tailwind CSS.
It is deployed to GitHub Pages using GitHub Actions.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Deployment

The site is automatically deployed to GitHub Pages via a GitHub Action defined in `.github/workflows/deploy.yml`.
Any push to the `main` branch will trigger a deployment.
